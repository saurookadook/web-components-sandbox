# web-component-sandbox

A little sandbox environment to play around with [Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) and related APIs.

## Setup

ðŸ™‚

```sh
$ nvm install $(cat .nvmrc)
$ npm install
```

## Starting the Server

**With Docker**

```sh
$ docker compose up -d
```

**Without Docker**

```sh
$ cd server
$ npm run start
```
